## Approach

- we get the LSB by n&1 , do a left shift to add this bit.
- add using res |= n&1 , and do a right shift to process next bit , do this for all the bits. 

---

## Code (Java)

```java
public int reverseBits(int n) {
	int res = 0;
	
	for(int i=0;i<32;i++)
	{
		res <<=1;
		res |= (n&1);
		n >>=1;
	}
	return res;
}

```
