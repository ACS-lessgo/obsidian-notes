## Approach

- XOR Properties 
	- XOR of a number with itself is always zero).
	- a ^ a = 0
	- XOR of a number with zero is the number itself.
	- a ^ 0 = a

---

## Code (Java)

```java
public int singleNumber(int[] nums) {

	int res = nums[0];
	
	for(int i=1;i<nums.length;i++)	
	{	
		res = res ^ nums[i];
	}
	return res;
}

```
