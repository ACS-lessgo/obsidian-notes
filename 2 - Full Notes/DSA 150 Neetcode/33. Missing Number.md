## Code (Java)

```java
public int missingNumber(int[] nums) {

	int res = nums.length;
	
	for(int i=0;i<nums.length;i++) res += i - nums[i];
	
	return res;
}

```

```java
public int missingNumber(int[] nums) {

	int res = nums.length;
	
	for(int i=0;i<nums.length;i++) res = res ^ i ^ nums[i];
	
	return res;
}

```
